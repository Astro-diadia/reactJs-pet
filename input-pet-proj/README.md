# инпуты react

  данное приложение будет с собой представлять анкету на 10-15 страниц,
  приложение будет запоминать данные и в конце выдавать cv работника


## Запуск

1. Установить зависимости:

    ```bash
    cd input-pet-proj
    npm install
    ```

2. Запустить приложение:

    ```bash
    cd input-pet-proj
    npm run dev
    ```

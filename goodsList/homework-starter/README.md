# Список покупок

    приложение реализует функционал списка покупок,
    используя react hooks

## Запуск

1. Установить зависимости:

    ```bash
    cd LogicInComponents/homework-starter
    npm install
    ```

2. Запустить приложение:

    ```bash
    cd LogicInComponents/homework-starter
    npm run dev
    ```

# Список покупок

    приложение реализует функционал списка покупок,
    используя react hooks(useState, useEffect, useRef) для запоминания,
    рендеринга, удаление и фокусирования элементов страницы

## Запуск

1. Установить зависимости:

    ```bash
    cd goodsList/homework-starter
    npm install
    ```

2. Запустить приложение:

    ```bash
    cd goodsList/homework-starter
    npm run dev
    ```

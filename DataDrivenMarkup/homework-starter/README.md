# Магазин чудес

    данное приложение имитирует в магазин,
    рендерит и показывает товары информация о которых получена из памяти,
    имеется система автоматического просчёта скидок

## Запуск

1. Установить зависимости:

    ```bash
    cd DataDrivenMarkup/homework-starter
    npm install
    ```

2. Запустить приложение:

    ```bash
    cd DataDrivenMarkup/homework-starter
    npm run dev
    ```
